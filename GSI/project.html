ErfanGSIs Project

The erfanGSIs project is made by Erfan Abdi and the tool makes it very easy for anyone to build GSIs. Initially the project was closed source but later erfan released the tool to the public so it can be improved with the help of other people. Till now there are several contributed to the project in many ways and all the information can be found on the github.


ErfanGSI tool can be used on Mac/Linux operating systems and you can build your own GSIs according to your needs.
