ErfanGSIs

These are the basic instructions for flashing an erfanGSI with a TWRP. All the erfanGSIs are compressed in a .7zip file, you need to extract the .IMG file from it using tool such as WinRAR on PC or Zarchiver on Android device.If you don't know how to flash a GSI in system partition ,then please read this article on XDA and check the "Flashing through TWRP" section.

Note - Always install google play services apk after booting, check download section for more information

Permissiver & Zygote Fix are not needed on any GSI which is built after 25th September. If you flashing any GSI which is built before this date then you have to use permissiver & Zygote Fix (For Android 10 - Aonly devices). AB devices never needs zygote fix.


on A-Only devices

    For Pie GSIs by Erfan

  1. Wipe Data/System
  2. Flash a treble ROM
  3. Flash the IMG in system partition
  4. Flashing Magisk is optional
  5. Reboot

For Q/Android 10 GSIs by Erfan

  1. Wipe Data/System
  2. Flash a treble ROM
  3. Flash the IMG in system partition
  4. Flashing Magisk is optional
  5. Reboot

on AB devices

    For Pie GSIs by Erfan

  1. Wipe Data/System
  2. Flash a treble ROM
  3. Flash the IMG in system partition
  4. Flashing Magisk is optional
  5. Reboot

For Flashing Q/Android 10 GSIs on AB devices - The instructions are same as flashing Pie GSI , you don't need Zygote fix for AB devices.

Error Flashing Zips

There are some errors which users get while flashing GSIs and Zips - So what are they ?

1. File is larger than target device - This means the size of device's system partition is lesser than the GSI image which are you trying to flash. In simple words this means you can't fit a pumpking in your coffee mug.

In this case the solution is resizing your system partition which can be dangerous if the institution not followed properly. It can damage your system and brick your phone. This is not suggested for people who don't know what they are going to do. Do not do thing on Moto devices. Here is link for a repartitioning a device - Resize System


2. Error when flashing permissiver- You can get errors sometimes when flashing permissiver which are related to ramdisk. The basic solution which I know is - Use Permissiver v4 first, if it gives error then try flashing permissiver v5. If v5 also gives errors then flash Magisk and try flashing permissiver V5 or V4 again. It can solve the issue sometimes if flashed after Magisk but is not guaranteed to work every time.
